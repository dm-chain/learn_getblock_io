## learn_getblock_io

## Prices

1. Loại `shared node`

Free 40k lượt request

Mua thêm lượt request:
* $6: 500k
* $10: 1M
* $30: 5M
* $200: 50M
* $500: unlimited requests

2. Loại `dedicated node`

Tạo node riêng, trả tiền tháng

https://getblock.io/en/dedicated-nodes/
